# Maillot Designer — Direction artistique & prompts IA

Skill de direction artistique pour concevoir des maillots de football créatifs et
rédiger les prompts IA correspondants. Construit à partir d'une bibliothèque de
références réelles (Nike, adidas, Jordan, Dior, Stüssy, Corteiz...) et de concepts
fan-made premium.

## Quand l'utiliser

- "crée-moi un maillot pour…" / "j'ai pas d'idées de design"
- "donne-moi un prompt pour Flow / Nano Banana / Midjourney"
- "vectorise ce maillot" / "fais une version concept de…"
- décliner un design en plusieurs coloris (home / away / third)

## Méthode en 4 étapes

Toujours suivre cet ordre. Ne JAMAIS sortir un prompt générique : passer par le
choix d'un **archétype** + **identité** + **palette** + **mise en scène**.

1. **Cadrer** — Demander (ou déduire) : équipe/club, archétype voulu, 1 contrainte
   forte (ex. "bleu + or", "streetwear", "rétro 90s"). Si l'utilisateur dit
   "surprends-moi", piocher un archétype et l'annoncer.
2. **Composer** — Assembler le concept à partir des briques ci-dessous
   (archétype → motif → coupe/col → palette → emblème → mise en scène).
3. **Décrire** — Rédiger le prompt avec le `TEMPLATE DE PROMPT`. Toujours fournir
   une version **anglaise** (meilleurs résultats IA) ET un résumé français.
4. **Décliner** — Proposer 1 variante alternative (autre coloris ou autre archétype)
   et demander s'il faut affiner.

## Les 10 archétypes (ADN visuel)

Chaque archétype = une direction. Toujours en nommer un explicitement.

1. **Couture / Jacquard royal** — motif damassé ou jacquard ton sur ton (fleur de lys,
   baroque, toile de Jouy), col polo, emblème brodé en relief or, palette mono + or.
   *Réf : Dior×Nike, France bleu jacquard, Espagne baroque.*
2. **Rétro 90s revisité** — col polo ou col V, piping incurvé sur le torse, mesh,
   bandeau dents-de-scie/zigzag, dégradé vertical. *Réf : Allemagne 90, Pays-Bas, Brésil.*
3. **Streetwear collab** — couleur tonale mono + texture embossée (léopard/tribal) +
   GROS logo script/graffiti central + 1 accent (or ou néon). *Réf : Stüssy, Corteiz.*
4. **Patrimoine culturel** — motif identitaire all-over racontant une culture
   (aztèque, maori/koru, fleur de lys, plumes de coq). *Réf : Mexique, NZ, France.*
5. **Rorschach / Symétrie miroir** — motif symétrique central type tâche d'encre,
   dégradé sombre, fines rayures. *Réf : France×Jordan bordeaux.*
6. **Dégradé chromatique** — dégradé fluide (vertical, diagonal "flou", ou flammes)
   + grille hexagonale optionnelle. *Réf : Chiefs, Holic, Milan sunset.*
7. **Organique 3D** — nervures/biomécanique en relief, effet matière, coupe oversize.
   *Réf : Milan Cl4e.*
8. **Bande centrale verticale** — bande contrastée verticale portant le numéro,
   épaules à bandes. *Réf : All Blacks concept.*
9. **Solaire / Rayons** — motif soleil en éventail + halos dégradés (aura rose/violet).
   *Réf : Uruguay×Stüssy.*
10. **Blokecore / Halftone** — trame de points (halftone), col rond insert V, gros
    wordmark minuscule, esthétique TV/rave 90s. *Réf : Vicinity TV.*

## Briques de composition

**Cols** : polo classique · polo contrasté (or/noir) · col V rétro · col rond insert-V ·
col à liseré tricolore intérieur · col ethnique brodé.

**Coupe** : moderne ajustée · oversize rétro · manches raglan.

**Structures** : piping incurvé sur le torse · panneaux latéraux en forme de carte/courbe ·
panneaux "flammes" latéraux · inserts sous les bras · épaules à bandes (3 stripes).

**Motifs / textures** :
- *Ton sur ton* : jacquard fleur de lys, damassé baroque, léopard embossé, tribal/koru,
  aztèque, toile de Jouy, géométrique tribal, coup de pinceau / vagues.
- *Graphiques* : dents-de-scie / zigzag, triangles dégradés, rayures verticales,
  rayures diagonales floues, halftone (trame de points), grille hexagonale, rayons solaires.
- *Relief* : nervures organiques 3D, broderie en relief.

**Emblèmes** : coq doré brodé + 2 étoiles + FFF · lion/aigle héraldique or · écusson rond ·
emblème encadré · symbole national (fleur de lys, étoiles championnes).

**Branding central** : wordmark club · logo script/graffiti surdimensionné · numéro géant.

## Palettes signature

- **France royale** : bleu profond `#1A2A52` / `#1B2C5A` + or `#C9A227` + blanc + touche rouge `#ED2939`.
- **France×Jordan luxe** : bordeaux `#5B1A23` + noir + or.
- **Streetwear dark** : noir + accent unique (blanc, néon ou or).
- **Sunset / flammes** : jaune `#FFC72C` → orange `#FF6A00` → rouge `#D7263D` → noir.
- **Baroque royal** : rouge profond + or + marine.
- **Kawaii** : rose dégradé / cyan / noir+rose (déclinaison personnage).

## Mise en scène (le décor RACONTE l'identité)

Toujours préciser la présentation — elle compte autant que le maillot :

- **Mockup technique vectoriel** : flat, avant + arrière, fond blanc pur, labels/codes
  produit (style "design sheet"). → pour le côté "vectorisé" demandé.
- **Sur cintre, fond thématique** : papier peint fleur de lys, bois sculpté tribal,
  velours, mur baroque, sakura + Mont Fuji, ciel coucher de soleil.
- **Éditorial / lookbook** : mannequin, lumière dramatique, décor cathédrale/pierre.
- **Atelier couture** : fils, machine à tricoter, tissu drapé (vibe luxe).
- **Planche multi-coloris** : 3 variantes avant/arrière alignées sur fond neutre.

## TEMPLATE DE PROMPT (anglais — à remplir)

```
[PRESENTATION], of a [CUT] football / soccer jersey, [VIEW].
[TEAM] kit, [ARCHETYPE] style.
Base color [COLOR1] [+ gradient to COLOR2]. [MOTIF] tone-on-tone across the body.
[COLLAR] collar [trim detail]. [SIDE PANELS / STRUCTURE]. [SLEEVE detail].
[EMBLEM] on the chest, [STARS], [CENTRAL BRANDING].
[COLOR ACCENTS on collar/cuffs/trim].
[SCENE / BACKGROUND].
Crisp clean lines, [flat vector OR photoreal fabric], high detail, symmetrical,
no people [or: editorial model shot], professional sportswear illustration.
```

Mots-clés qui forcent un rendu **vectoriel** : `flat vector, no shadows, white background,
technical drawing, sharp edges`.
Mots-clés qui forcent un rendu **réaliste/couture** : `photoreal, fabric texture, embossed
jacquard, soft studio lighting, 8k`.

## Exemple complet (France — Couture jacquard)

> "Photoreal product shot on a hanger against a royal blue fleur-de-lys wallpaper
> background, of a slim-fit football jersey. France national team kit, couture jacquard
> royal style. Deep navy blue base (#1A2A52) with a tone-on-tone fleur-de-lys and baroque
> swirl jacquard texture across the body. White polo collar with an inner tricolor band.
> Curved gold piping on the chest, subtle red trim on the cuffs. Gold embroidered raised
> rooster crest with two gold champion stars and 'FFF'. Gradient tricolor swoosh.
> Soft studio lighting, embossed fabric texture, high detail, symmetrical, no people,
> professional sportswear illustration, 8k."

*Résumé FR :* maillot France couture, bleu marine + jacquard fleur de lys ton sur ton,
col polo blanc, piping or, coq doré en relief, présentation sur cintre fond royal.

## MODULE SURVÊTEMENT / TECHWEAR (vestes & ensembles)

Pour tout survêtement, tracksuit, veste, ensemble (haut + bas). Même méthode en
4 étapes, mais avec ces briques spécifiques.

### Archétypes survêtement

1. **Techwear flammes** — veste cropped à capuche noire, panels incurvés en dégradé
   (orange→rouge "flammes", ou bleu) suivant des coutures ergonomiques, mesh gris,
   logo croix `+` minimal, cordons de serrage. *Réf : Night Lab cropped set.*
2. **Blokecore halftone** — track jacket noire, graphique en trame de points
   (halftone) façon flamme/tribal, panels gris réfléchissants, col montant zippé.
   *Réf : Night Lab halftone.*
3. **Techwear organique / shell** — veste shell à capuche, panels biomécaniques
   brillants (bleu/argent) en relief, attaches techniques, vibe outdoor futuriste.
4. **Varsity wavy / luxe** — veste en cuir, color-block ondulé (noir/crème), broderie
   script en relief ("Night"), logo croix, coupe oversize. *Réf : Night Lab leather.*
5. **Total look mono** — ensemble veste + pantalon baggy ton sur ton (souvent noir),
   1 seul graphique signature, mise en scène lookbook studio.

### Briques survêtement

**Pièces** : veste à capuche cropped · track jacket col montant · shell jacket ·
veste varsity cuir · short assorti · pantalon baggy / parachute · bonnet (set complet).

**Cols / fermetures** : col montant zippé · capuche à cordons · zip central full-length ·
poches zippées inclinées · ourlet à cordon élastique.

**Structures / coutures** : **panels ergonomiques incurvés** (signature techwear) ·
color-block ondulé (wavy) · empiècements épaules · inserts mesh.

**Motifs / textures** : dégradé "flammes" sur panels · halftone (trame de points) ·
panels réfléchissants 3M · biomécanique brillante · cuir mat.

**Branding** : logo croix `+` minimal · petit swoosh/logo brodé poitrine · script
brodé dans le dos · code discret (ex. "NL 25") en bas de dos.

**Palettes** : noir + 1 accent (flammes orange-rouge / bleu électrique / crème-jaune).
Garder très sobre : noir dominant, 1 zone de couleur.

### Mise en scène survêtement

- **À plat sur béton / canapé** (flat lay produit, vibe Vinted/lookbook).
- **Lookbook studio** : mannequin, fond gris/beige neutre, lumière douce, total look.
- **Set avant + arrière** posé à plat (montre le graphique dos).

### Exemple complet (Survêtement France — Techwear flammes tricolore)

> "Product flat lay on a raw concrete floor, of a cropped hooded techwear tracksuit set
> (jacket + matching shorts). Black base with ergonomic curved panels in a blue-white-red
> tricolor gradient running down the chest and sleeves, grey mesh accents, full-length
> central zip. Minimal white cross '+' logo on the chest, small embroidered rooster crest.
> Drawcord elastic hem. Matching shorts with the same curved tricolor panels. Photoreal
> fabric, technical sportswear, soft even lighting, high detail, no people, 8k."

*Résumé FR :* survêtement France techwear, noir + panels tricolores en dégradé suivant
des coutures ergonomiques, logo croix, coq brodé, ensemble veste + short, flat lay béton.

## Règles de qualité

- Toujours nommer l'archétype choisi au début de la réponse.
- Toujours donner le prompt EN anglais + un résumé FR + les codes couleur HEX.
- Toujours proposer 1 variante (autre coloris OU autre archétype).
- Pour "vectoriser" : privilégier mockup flat avant+arrière, fond blanc, mots-clés vecteur.
- Éviter les marques déposées dans le prompt final si l'usage est commercial
  (remplacer "Nike/adidas swoosh" par "generic sportswear logo placeholder").
- Garder la cohérence : 1 couleur dominante + 1 à 2 accents max.
